How to Edit Figma Files?

In order to edit the file, you need to have Adobe Figma installed on your system, and the fonts mentioned in the credits installed.


As you can see on attached picture, the layers are well organized and grouped into folders, which make editing very easy.
So, all you have to do, is select the layer, and then edit it with Figma .
That's all! Enjoy.

How to Change Image

Drop your image to page, drag your picture above the later with placeholders(usually they are named Mask Image, bg, image ), right click on your image layer -> Create Clipping Mask or use shortcuts Control + Alt + G for Windows and Command + Option + G for Mac

Empty Figma Files?

To save on upload capacity, I started conducting hide entire Groups when i save some new Figma Files.

How you can turn on Group and view all Layers?

It is super easy. You need to open Photoshop. Go to Windown > Layers (F7) to show the Layers Panel.
In The Layers Panel, Click The Eye Icon. You will turn on Groups of Layers. Enjoy.

Changing Font

Mediclinis uses Font family: Heebo as a default font and Poppins as a Heading Font from the Google Fonts Library. To change the Font, you will need to Edit the Bottom Link with your Custom Font link in style.css file and change the body css font-family

If you plan to use a Self Hosted font. Here is an Example for using Self Hosted Font.

    @import url("https://fonts.googleapis.com/css2?family=Heebo:wght@400;500;600;700;800;900&display=swap");

    body {
        font-family: "Heebo", sans-serif
    }

    h1,h2,h3,h4,h5,h6 {
        font-family : "Heebo", sans-serif
    }
